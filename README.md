# learning-streak-tracker
learning streak tracker for daily streak monitoring system.
Project: Daily Learning Streak Tracker

Features:
- Mark "I Studied Today"
- Current streak calculation
- Study history page
- Prevent duplicate entries
- Next.js API routes

Tech Stack:
Next.js
TypeScript
TailwindCSS
API Routes

How Streak Logic Works:
If user studies consecutive days → streak increases.
If a day is skipped → streak resets to 1.
